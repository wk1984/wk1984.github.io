The original model was developed by Ling Feng and Zhang Tingjun in 2004.

The model is simplied to 1D heat tranfer model, excluding the surface balance process.

It has been compiled with a python interface.

This version passed the test in Mac OS with python 2.7 and gfotran 6.1.

Ling F, Zhang T. 2004. A numerical model for surface energy balance and thermal regime of the active layer and permafrost containing unfrozen water. Cold Regions Science and Technology 38(1): 1–15.

def cal_ALT(x, y):
    
    import numpy as np
    
    idx_pos = np.where(y>=0)
    idx_pos = idx_pos[-1][-1]
    
    y0 = y[idx_pos]
    y1 = y[idx_pos+1]
    
    x0 = x[idx_pos]
    x1 = x[idx_pos+1]
    
    diy = y0-y1;
    dix = x0-x1;
    
    ALT = x0 - dix/diy * y0;
    
    return ALT
